# Clawdbot Architecture Docs

[Clawdbot](https://github.com/clawdbot/clawdbot) のアーキテクチャ解析ドキュメントです。

## 概要

Clawdbotは、複数のメッセージングプラットフォーム（Discord、Slack、Telegram、WhatsAppなど）とAIエージェントを接続するためのフレームワークです。

このリポジトリでは、Clawdbotのアーキテクチャを解析して、わかりやすく可視化したドキュメントを提供しています。
